HTML

  • HTML is short for HypterText Markup Language.
  • It is the standard markup language for documents designed to be displayed in a web browser.
  • Semantic tags allow screen readers to determine where different elements are on a web page and how they're formatted.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is short for Cascading Style Sheets.
  • It is a style sheet language used for describing the presentation of a document written in HTML.
  • It is used to style and layout for web pages.
  • Some examples are to alter the font, color, size, and spacing of content.
  • It is also used to split content into multiple columns, add animation and other decorative features.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is short for Global Information Tracker.
  • It is a free and open source distributed version control system.
  • It is designed to handle everything from small to very large projects with speed and efficiency.
  • Some features are local branching, convenient staging areas, and multiple workflows.
  • It allows you to manage and keep track of your source code history.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a programming language that is one of the core technologies of the world wide web.
  • It is used to create dynamic and interactive web content like applications and browsers.
  • There are three main types: primitive (primary), composite (reference), and special data types.
  • String, Number, and Boolean are primitive data types.
  • Object, Array, and Function are composite data types.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
